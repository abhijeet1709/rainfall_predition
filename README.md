# 🌧️ Rainfall Prediction App

A Machine Learning-based web application that predicts whether rainfall will occur based on key weather parameters. The app provides an interactive interface where users can input real-time data and get instant predictions.

---

## 🚀 Features

* Predict rainfall using ML model
* Interactive UI built with Streamlit
* Real-time user input
* Fast and lightweight
* Easy to run and deploy

---

## 📊 Input Parameters

The model takes the following inputs:

* Pressure
* Dew Point
* Humidity
* Cloud Coverage
* Sunshine Duration
* Wind Direction
* Wind Speed

## ⚙️ Setup & Installation

### 1️⃣ Clone the Repository

```
git clone https://github.com/your-username/rainfall-prediction.git
cd rainfall-prediction
```

---

### 2️⃣ Create Virtual Environment (Recommended)

```
python -m venv myenv
```

---

### 3️⃣ Activate Virtual Environment

**For Windows (PowerShell):**

```
myenv\Scripts\activate
```

**For Mac/Linux:**

```
source myenv/bin/activate
```

---

### 4️⃣ Install Dependencies

```
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```
streamlit run app.py
```

Then open the local URL shown in your terminal (usually `http://localhost:8501`)
